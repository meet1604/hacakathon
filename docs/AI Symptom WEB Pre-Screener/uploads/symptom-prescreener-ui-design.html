    Responsive Design Guidelines

    Mobile (< 640px)

    • Single column layout
    • Full-width cards
    • Stacked stat cards
    • Bottom navigation
    • Collapsible sections

    Tablet (640px - 1024px)

    • 2-column grid
    • Side navigation drawer
    • Expanded chart views
    • Side-by-side comparisons

    Desktop (> 1024px)

    • 3-column layouts
    • Fixed sidebar navigation
    • Multi-panel dashboards
    • Hover interactions
    • Keyboard shortcuts
